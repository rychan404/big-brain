## Variable Syntax

```python
variable_name = value
```

## Type Conversion

`type()` 
- returns data type

```python
type('foo') # returns <class 'str'>
```

`float()` 
- converts str/int to float
- `float('inf')`
	- represents positive infinity

`int()`
- converts to int

`list()`
- converts to list from iterable
## Integers

`abs()` 
- returns absolute value of num

`sum(1, 2, 3)` 
- returns sum of items

`max(1, 2, 3)` 
- returns largest input value
